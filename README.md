# MagicBlack

A magic plugin for the Decky Loader that overlays the screen with black color, emulating the screen being turned off on the Steam Deck OLED.

### Turning Off the Screen

- To turn off the OLED screen, enable the `Black Background` switch â€” a fully black overlay will be applied to the display. On an OLED panel, each pixel emits its own light, and black means the pixel is not emitting anything, so the screen effectively turns off.
- To bring the image back, press <img src="./docs/images/dark/qam.svg#gh-light-mode-only" height=16><img src="./docs/images/light/qam.svg#gh-dark-mode-only" height=16> to return to the plugin and disable the `Black Background` switch again.

### Reducing Flicker

- To reduce OLED flicker, set the screen brightness to 100%. Then enable the black background and adjust its transparency using the `Opacity` slider to find a comfortable level.
